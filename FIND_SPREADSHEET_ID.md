# How to Find Your Google Sheets Spreadsheet ID

## Quick Method

1. **Open your Google Sheet** in a web browser
2. **Look at the URL** in the address bar
3. **Find the long string** between `/spreadsheets/d/` and `/edit`

## Example URLs

### Example 1:
```
https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms/edit
```
**Spreadsheet ID:** `1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms`

### Example 2:
```
https://docs.google.com/spreadsheets/d/abc123xyz789/edit#gid=0
```
**Spreadsheet ID:** `abc123xyz789`

### Example 3:
```
https://docs.google.com/spreadsheets/d/1a2b3c4d5e6f7g8h9i0j/edit?usp=sharing
```
**Spreadsheet ID:** `1a2b3c4d5e6f7g8h9i0j`

## What It Looks Like

- Usually **44 characters long**
- Contains **letters, numbers, and dashes**
- Looks like: `1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms`

## Still Can't Find It?

**Option 1:** If you don't have a Google Sheet yet:
- Create a new one at https://sheets.google.com
- The ID will be in the URL once you create it

**Option 2:** If you're viewing it in a mobile app:
- Open it in a web browser instead
- The URL will be visible there

**Option 3:** Share the sheet with yourself:
- Click "Share" button
- Copy the sharing link
- The ID is in that link too!
